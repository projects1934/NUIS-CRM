## ⚠️ ניקוי אצלך — רק אחרי שאישר שזה רץ אצלו
כשהוא מאשר שהאתר עלה ועובד על ה‑Netlify + Supabase שלו:

1. **בטל לו את הגישה ל‑Supabase שלך** — Supabase → הארגון שלך → Team/Members → הסר אותו. (וגם כל שיתוף ברמת הפרויקט, אם יש.)
2. **מחק את הפרויקט `datanuiscrm`** מה‑Supabase שלך — Project Settings → General → Delete project. (זה מוחק סופית — רק אחרי שאתה בטוח שהעותק שלו עובד.)
3. **הסיסמה של מסד הנתונים** (`yDYQ5lGP2PAoKPXW`) — הקלדת אותה כאן בצ'אט. אחרי שתמחק את הפרויקט היא ממילא לא רלוונטית. אם תחליט לא למחוק — אפס אותה: Database → Settings → Reset password.
4. **Vercel** — אפשר למחוק את הפרויקט הישן מ‑Vercel (הוא כבר לא בשימוש).

בזה זה "יורד ממך" לגמרי.
