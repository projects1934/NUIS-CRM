# Nuiscrm — העברת הפרויקט אליך (קוד + דאטה + פריסה ל‑Netlify)

בחבילה הזו יש שני קבצים:
1. **`Nuiscrm-netlify-ready.zip`** — כל קוד האפליקציה, כבר מותאם ל‑Netlify.
2. **`nuiscrm_full_migration.sql`** — כל הסכימה + כל הנתונים של מסד הנתונים (201 שותפים, 19 פרויקטים, משתמשים, הודעות וכו').

צריך 3 שלבים. הכול על החשבונות שלך.

---

## שלב 1 — מסד הנתונים (Supabase)

1. היכנס ל‑Supabase שלך → פתח את הפרויקט שלך (או צור פרויקט חדש וריק).
2. בתפריט השמאלי: **SQL Editor** → **New query**.
3. פתח את `nuiscrm_full_migration.sql`, העתק את **כל** התוכן, הדבק בעורך, ולחץ **Run**.
   - זה יוצר את כל הטבלאות וטוען את כל הנתונים. ריצה אחת, מספר שניות.
   - הקובץ בטוח להרצה על פרויקט חדש וריק (עשינו לזה בדיקת טעינה מלאה מראש).
4. קח את מחרוזת החיבור: למעלה **Connect** → בחר **Session pooler** → העתק את ה‑URI.
   - הוא נראה כך: `postgresql://postgres.XXXX:[YOUR-PASSWORD]@aws-...pooler.supabase.com:5432/postgres`
   - החלף את `[YOUR-PASSWORD]` בסיסמת מסד הנתונים **שלך**. שמור את המחרוזת לשלב 3.
   - חשוב: **Session pooler** (פורט 5432), לא Transaction pooler.

---

## שלב 2 — הקוד (GitHub)

1. פרוס (unzip) את `Nuiscrm-netlify-ready.zip`.
2. צור ריפו חדש ב‑GitHub שלך והעלה אליו את התוכן (או `git init && git add . && git commit && git push`).
   - אם כבר העתקת את הריפו הישן — **החלף אותו בגרסה הזו**, כי רק בה יש את קבצי ה‑Netlify (`netlify.toml`, `netlify/functions/api.js`).

---

## שלב 3 — הפריסה (Netlify)

1. ב‑Netlify שלך: **Add new site** → **Import an existing project** → חבר את GitHub ובחר את הריפו.
2. אל תיגע בהגדרות ה‑build — Netlify קורא אותן אוטומטית מ‑`netlify.toml`.
3. לפני הפריסה, הוסף משתנה סביבה אחד:
   - **Key:** `DATABASE_URL`
   - **Value:** מחרוזת החיבור מהשלב הקודם (Session pooler, עם הסיסמה שלך).
4. לחץ **Deploy**.

זהו. האתר יעלה בכתובת `https://<שם-האתר>.netlify.app`.

---

## התחברות לאפליקציה

המשתמשים הקיימים הועברו יחד עם הקוד הסודי (PIN) שלהם — מתחברים עם אותם שם‑משתמש ו‑PIN כמו קודם.
(אם תרצה משתמש‑אדמין חדש: הוסף ב‑Netlify גם `DEFAULT_ADMIN_USER` ו‑`DEFAULT_ADMIN_PIN` — אבל זה לא חובה.)

## אם ה‑build נכשל
נסה להוסיף ב‑Netlify משתנה סביבה `NODE_VERSION` = `20`.

## מה האפליקציה צריכה (סיכום משתני סביבה)
- **`DATABASE_URL`** — חובה (Supabase Session pooler שלך).
- השאר אופציונלי, יש ברירות מחדל.
